Bestiario Recursos Equipamiento

Recursos

Estos son materiales necesarios para mejorar el equipamiento o crear objetos y hay de tres tipos

Recursos Naturales

Estos se consiguen por el mapa y se usan normalmente para trampas y pociones

Recursos Animales

Estos se consiguen al despellejar un monstruo o bien capturandolo. Se usan para mejorar el equipamiento

Recursos Minerales

Estos se consiguen picar un mena y sirven para subir de nivel el equipamiento
